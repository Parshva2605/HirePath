================================================================================
                        KNOWN LIMITATIONS
================================================================================

1. GitHub Analysis
   - Requires Ollama to be running
   - Limited to 100 public repos per user
   - No private repository access (by design)

2. Job Matching
   - Skills-based matching (no NLP without ML models)
   - Limited to local job database
   - No real-time job board integration

3. Resume Parsing
   - Best with standard resume formats
   - May struggle with unusual layouts/fonts
   - PDF extraction depends on file quality

4. Ollama Requirements
   - Requires ~2GB disk space for models
   - Requires 2GB+ RAM for inference
   - First inference slower (model loading)

5. API Rate Limits
   - GitHub public API: 60 requests/hour per IP
   - No rate limiting on FastAPI (implement as needed)

                      DEPLOYMENT INSTRUCTIONS
================================================================================

DEVELOPMENT:
  1. Run start_backend.bat (Windows)
  2. Or: cd agent && venv\Scripts\activate && uvicorn main:app --reload
  3. API available at http://localhost:8000

PRODUCTION:
  1. Install gunicorn: pip install gunicorn
  2. Run: gunicorn -w 4 -b 0.0.0.0:8000 agent.main:app
  3. Configure reverse proxy (nginx)
  4. Set environment variables from .env file

DOCKER:
  1. Build: docker build -t hirepath .
  2. Run: docker run -p 8000:8000 hirepath
  3. Ollama: docker run -p 11434:11434 ollama/ollama

================================================================================
                         TROUBLESHOOTING
================================================================================

Issue:              Solution:
─────────────────────────────────────────────────────────────────────────────
Ollama not found   → Download from https://ollama.ai and install
Can't import       → pip install -r agent/requirements.txt in venv
Port 8000 in use   → Kill process: netstat -ano | findstr :8000
GitHub fails       → Check internet connection, GitHub might be down
Resume parsing     → Ensure file is actual PDF/DOCX, not corrupted
Spacy model        → python -m spacy download en_core_web_sm
